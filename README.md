# Solid Ops — SketchUp Plugin

Plugin d'opérations booléennes sur les solides pour SketchUp (version gratuite).

## Fonctionnalités

- **Union** — Fusion de plusieurs solides en préservant les vides internes
- **Subtract** — Soustraction interactive : clic sur la base, puis clics successifs sur les outils à soustraire
- **Split** — Découpe de deux solides en 3 parties (A-B, B-A, A∩B)

Accessible via **Extensions > Solid Ops** ou la barre d'outils.

## Installation

1. Télécharger `build/dro_solid_ops.rbz`
2. Dans SketchUp : **Window > Extension Manager > Install Extension**
3. Sélectionner le fichier `.rbz`
